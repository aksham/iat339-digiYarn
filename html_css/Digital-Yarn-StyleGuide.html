Digital Yarn Style Guide

About

Digital Yarn is a mobile and PC indie game developer. They plan to use their site not only as a showcase of games they've created, but also to convert it into a platform to promote and sell games and related merchandise. While the website will mostly have graphics and images as its main visual elements in order to showcase the games, underneath those images, this is what it'll look like.

Colors What unites the visual style

The colors used here are in a hierarchy of sorts. For the branding colors, the blue is meant to complement the pink as much as much as it can follow the rules of element hierarchy at the same time. Green is to be sparingly used, and is mainly intended for extraneous and/or miscellaneous elements.

Body, text colors: Things to read, and things to read on

  • rgba(30, 35, 35, 1.0)

    The main background color

  • rgba(20, 25, 25, 1.0)

    Used for subtle highlighting

  • rgba(255, 255, 255, 1.0)

    Used for body text

  • rgba(135, 135, 135, 1.0)

    Used for subtext

Branding colors: This means Digital Yarn

  • rgba(255, 110, 0, 1.0)

    Key branding highlight

  • rgba(50, 169, 247, 1.0)

    Secondary highlight

  • rgba(100, 200, 40, 1.0)

    Tertiary highlight

Images The visuals

The images on Digital Yarn are generally intended to be full bleed backgrounds, and other images that are embedded will most likely break free from the grid itself (characters floating around, with animations). As such, the only image styles to be used are to ensure that there are no extraneous elements being added onto the images, such as borders that may be added on by default, if the image is a link.

Example:

CSS Selector:

img { border: none; }

Sample HTML:

<a href="#images"><img src="img/cat_with_yarn.jpg"></a>
